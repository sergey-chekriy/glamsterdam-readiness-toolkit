# Glamsterdam Readiness Toolkit

Open-source CLI and Node.js library for analyzing Solidity projects for Glamsterdam upgrade readiness.

## Packages

- `@grt/core` - project ingestion, report model, analyzer orchestration
- `@grt/rules` - Glamsterdam readiness rules
- `@grt/fixtures` - test fixtures and sample projects
- `@grt/cli` - command-line interface

## Quick start

```bash
npm install
npm run build
npm run test
npm run dev -- scan ./examples/sample-foundry
```
